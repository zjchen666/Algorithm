#### 序列型动态规划
   不带状态： 一般是 f[i] = max(f[i] + k, f[i-j] ) |  i > j >= 0
   带状态：二维数组，k种状态： f[i][k] = max(f[i-1][k], f[i][k-1])
   处理边界条件时， 可能需要多分配一个元素。
   f为二维数组时， 优化空间时可以使用滚动数组。
   
#### 坐标型动态规划
   一般不需要多分配元素，f 为 m * n 矩阵.
